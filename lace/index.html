Language Agnostic Coding Event

Published on 12 PM Friday 04 October 2013

code

Language Agnostic Coding event was the name of an online coding event that I organized as a part of Freshers orientation in our college. It consisted of solving 5 questions in 12 hours online, I used python on appengine to make it. Here is the site

the source code of the working site can be found here on github

There is a limit on the reading quota of appengine which is 0.05 million reading requests on database per day, this limit ran out and I had to close the event early. even though there were only 300 submissions on the site over the whole period of the event, the limit ran out since I was querying the database on get requests too and there were five such pages. And each get and post requests comprises on many smaller requests [link to that question on appengine]

Well overall it was a very rich learning experience for me, coming up with 5 original questions in one night.

Actually I had decided to put projecteuler questions as it is on the site, but then some students started googling the answers, I had to postpone the event and put original questions instead. One other thing that was a challenge was to query the database, though the documentation of appengine is quite thorough, but there are very few examples to work ou. I had to read a book on appengine to understand appengine more thoroughly.

It was my second project to be used by so many people, 50 participants and 300 submissions, first one is colorfox [link to google chrome app].

A lot of time we tend to underestimate certain things, and we don't know about what we don't know until we challenge ourselves. So periodic challenges are important for perpetual growth. Only then and only then do we try to surpass ourselves and grow. And growth is very important.

I wish to make a legacy out of lace, a bi-annual core coding event, which is not about language dependent things like pointers and trees and stuff, but about coding and maths, about things that matter, about things that are more universal.This is the change I would like to see in my college.

If you would like to be informed about the next time when lace is organized, stay tuned to the facebook page
