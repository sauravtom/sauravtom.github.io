On Blogging

Published on 12 PM Tuesday 18 March 2014

general

I have been blogging since the last two years, its been on and off mostly, but now that I've bought a dedicated domain name, I must get serious, how else am I suppose to justify 600 bucks(Rs) a year.

Earlier I used to blog on blogspot, here is the link, its full of nostalgia. But when I started getting serious about programming, I decided to make my own static pages blog, and found out that making it was real easy using python and flask.

I write the posts in the markdown format in vim, and run a flask app which freezes them to a static webpages using Jinja Templating system, and then I cd to that directory and push that directory on github.
